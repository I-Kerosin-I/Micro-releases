#  Micro
Micro — это эмулятор микропрограммируемого процессора Intel 8086.<br/>
Данный проект является учебным. Он был создан в первую очередь для изучения языка C# и фреймворка WPF, а также для того, чтобы заменить устаревшее ПО, используемое для выполнения лабораторных работ по дисциплине ЭВМиПУ. <br/>
Исходный код программы является закрытым. Данный репозиторий служит только для распространения обновлений.
